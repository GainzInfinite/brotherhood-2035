# Brotherhood 2035

Elite command center for total life optimization. A dark, masculine self-improvement dashboard for tracking fitness, wealth, habits, journaling, and mindset.

## Tech Stack

- **Next.js 14** (App Router)
- **React 18**
- **TypeScript 5**
- **Tailwind CSS 3**
- **shadcn/ui** components

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the application.

### Build for Production

```bash
npm run build
npm start
```

## Project Structure

```
app/
├── layout.tsx              # Root layout with Sidebar and HeaderBar
├── page.tsx                # Home page
├── command-center/         # Main dashboard
├── health/                 # Fitness & nutrition tracking
├── wealth/                 # Financial tracking
├── mind/                   # Journal & mindfulness
├── check-in/               # Daily accountability
├── challenges/             # Goals & challenges
├── achievements/           # Badges & milestones
├── profile/                # User profile
├── clans/                  # Team features
├── analytics/              # Performance insights
├── settings/               # App settings
└── fatherhood/             # Optional family module

components/
├── Sidebar.tsx             # Left navigation
├── HeaderBar.tsx           # Top header with prestige info
├── PrimaryCard.tsx         # Main dashboard cards
└── SecondaryCard.tsx       # Secondary info cards
```

## Features

### Navigation
- Fixed left sidebar with all module links
- Active route highlighting
- Icon-based navigation with Lucide icons

### Layout
- Dark masculine theme (deep navy/charcoal with gold accents)
- Responsive design
- Fixed sidebar with scrollable main content
- Top header bar with user prestige/status

### Current Status
All routes are set up with placeholder pages. Ready for feature implementation.

## Theme

The application uses a carefully crafted dark theme:
- **Background**: Deep charcoal (#0a0e27)
- **Cards**: Navy with subtle gradients
- **Accent**: Gold (#eab308)
- **Text**: Light foreground for contrast

## Next Steps

Each module is ready for implementation. The structure supports:
- Progress tracking with percentages (not raw numbers)
- Visual indicators and progress bars
- Card-based layouts for dashboard tiles
- Consistent dark, masculine aesthetic throughout
